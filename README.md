The onu-status API is a reference design REST API that could be implemented as a northbound interface on a PON controller to provide read-only access to ONU identification, status monitoring, and performance management elements.

The .yaml file is the API definition itself, and the .html file provides a swagger interface for exploring the API.

The context of this API is the management of ONUs based on ITU-T PON technologies such as XGS-PON, 25GS-PON, and HS-PON.
