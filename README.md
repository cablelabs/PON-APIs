This respository contains REST APIs intended for the management of network equipment based on 
ITU-T PON technologies such as XGS-PON, 25GS-PON, and HS-PON. 

	ONU Status API:
	
	The onu-status API is a reference design REST API that could be implemented as a northbound
	interface on a PON controller to provide read-only access to ONU identification, status
	monitoring, and performance management elements.

	The onu-status.yaml file is the API definition itself, and the onu-status.html file provides
	a swagger interface for exploring the API.
